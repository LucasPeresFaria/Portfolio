Projetos

Samp - Sistema de Atribuição de Mapeamento de Aula (2014-2015)


O projeto baseia-se em atribuição de aulas que pode ser descrito por atribuir os docentes ao(s) curso(s) definidos em sua carga horária de trabalho.

Otimizar, agilizar e facilitar. Esses sao os principais obtivos desse projeto, que são os principais ojetivos desse projeto, que vão ajudar os coordenadores e os administradores da ETEC Jorge Street a atribuir um plano diretório de aulas aos professores de uma forma fácil e rápido.

O desenvolvimento do projeto foi utilizado a linguagem de programação C# e MySQL Server.


SBCNet - Sistema de gerenciamento de seleção e recrutamento para RH (2018-até o momento)


O sistema que será criado fará parte de todo o processo de recrutamento e seleção, com o objetivo de informatizar e oferecer suporte as etapas do processo que na sua maioria são feitas manualmente. O sistema será desenvolvido para a plataforma Web/Mobile, com o objetivo de unir os recursos disponíveis em ambas as plataformas para uma melhor experiência, tanto para o candidato quanto para o usuário do sistema.

O desenvolvimento do projeto será utilizado a linguagem de HTML5, CSS3, JavaScrit na aplicação web. E para os dispostivos mobiles a aplicação será desenvolvida com a linguagem Dart.
